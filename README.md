# Habit-Tracker
A full-stack habit tracking web application built with Django that helps users build consistency by tracking daily habits, visualizing streaks, and monitoring 30-day progress.

# Features
- User Authentication — Secure login/logout. Every user sees only their own habits.
- Full CRUD — Add, edit, delete habits anytime.
- One-Click Toggle — Mark habits as done/not done directly from the dashboard.
-  Streak Calculation — Automatic consecutive day streak tracking per habit.
- 30-Day Progress View — Visual grid showing completion history for the last 30 days.
- REST API — /api/habits/ endpoint built with Django REST Framework for future mobile/frontend integration.
- Django Admin — Full admin panel for database management during development.

# Tech Stack
Django, SQLite, Django Templates & rest framework
